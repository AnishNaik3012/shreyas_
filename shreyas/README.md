# Shreyas Module Export

This folder contains the code collected for:

- Prescription and OCR/report flow
- Nurse module
- Frontend source

## Contents

- `backend/` - FastAPI application source and dependency files needed by the selected modules.
- `frontend/` - Next.js frontend source, public assets, and project config.

Generated/runtime files were intentionally excluded, including virtual environments, `node_modules`, `.next`, `.git`, `.env*`, SQLite database files, uploads, and Python bytecode caches.
